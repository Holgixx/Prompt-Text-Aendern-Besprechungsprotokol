# Besprechungsprotokoll

## Vorlage fuer eine Besprechungszusammenfassung

### Zweck und Kontext
Kurze Erlaeuterung, warum das Treffen abgehalten wurde.

### Wichtige Entscheidungen
- Entscheidung A - Eigentuemer
- Entscheidung B - Eigentuemer

### Aktionspunkte
- Aufgabe 1 -> Verantwortlicher, Frist, Abhaengigkeiten
- Aufgabe 2 -> Verantwortlicher, Frist, Abhaengigkeiten

### Risiken und Hindernisse
- Ungeloestes Problem X
- Abhaengigkeit von Y

### Erwaehnte Ressourcen
- Dokumentenlink, Datensatz, externer Partner

### Naechste Schritte
- Folgetreffen am [Datum]
- Uebergabe der Ergebnisse
